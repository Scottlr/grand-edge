# Content Layers

Grand Edge uses three teaching layers for user-facing copy:

## 1. Simple layer

Use plain labels in the main interface first. The user should be able to act
without knowing trading or model vocabulary.

Examples:

- `Trade realism`
- `Expected profit`
- `What else may be affected`

## 2. Explanation layer

Use tooltips and short inline explainers when a user needs a little more
context without leaving the current workflow.

Examples:

- `TooltipTerm`
- one-sentence field help
- short glossary quick definitions

## 3. Learn layer

Use learn modals and advanced panels for deeper teaching, examples, and
technical vocabulary.

Examples:

- `LearnModal`
- `AdvancedTermPanel`
- glossary advanced notes

## Rule of thumb

The default UI must stay plain even when advanced detail exists. Technical
terms belong in explanation and learn surfaces, not in the first sentence a
user sees.
