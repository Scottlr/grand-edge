# Corpus Map

- `mechanical_relation_edge`: item sets, recipes, repairs, alchemy, dose/decant, charge/degrade.
- `market_event_context`: updates, events, bot-ban waves, item sinks, activity changes.
- `competitor_capability_context`: product gap analysis only.
- `source_policy_review`: licensing, summaries, copied-text risk, content hashes.
- `linked_item_reasoning`: graph reason atoms and Linked Items UI context.
- `blast_radius_context`: event or item shock scenario context.
- `ml_graph_feature_context`: graph feature/export naming and no-lookahead constraints.
