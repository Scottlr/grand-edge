# Market Intelligence Corpus

The planned market-intelligence corpus under `data/corpus/` stores sourced,
reviewable context for event nodes and graph-linked reasoning.

Expected files include:

- `source_registry.v1.json`
- `market_analysis.v1.json`
- `events.v1.json`
- `competitor_capabilities.v1.json`
- `review_notes.v1.json`

Use this corpus for updates, event hypotheses, market-analysis summaries, and
competitor gap notes. Treat every entry as context that still needs graph,
price, execution, and recommendation evidence before it influences user-facing
advice.
