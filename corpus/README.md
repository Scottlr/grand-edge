# Corpus

This folder holds source evidence for GrandEdge. Keep evidence grouped by type so implementation tasks can cite stable sources without mixing model notes, API observations, market samples, evaluation outputs, and product/design records.

The feature task plan remains the implementation source of truth. Corpus documents are supporting evidence: preserve provenance, split large inputs into reviewable files, and add a short index when importing a new evidence set.

## Current Evidence Groups

- `model-design/`: mathematical, statistical, and model-contract evidence for feature generation, predictions, recommendations, simulation, and metrics.
